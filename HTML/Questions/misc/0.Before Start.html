You will need to have the following information at hand to complete this report:
  • Information on all Wellcome awards (active and closed) (see list of awards attached in e-mail) 

  • Part 1 - Details of (for Wellcome awards only):
    • Invention disclosures - including the number of disclosures received
    • Patent applications filed
    • Granted patents - including the total number of granted patents (treat patent families as 1)

    • IP and agreements abandonment - including the number of agreements abandoned

  • Part 2 - Financial Reporting:
    • Wellcome awards which have received follow-on and/or leveraged funding and the amount in pounds (£).
    • Cumulative income (e.g. gross, net) breakdowns for all Wellcome-funded IP transactions that have generated revenue between ${e://Field/Reporting%20period} ${e://Field/Previous%20year}
